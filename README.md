# kotlin_keyword
for SAKURA Editor

ファイルをダウンロードしてサクラエディタのフォルダにコピーして下さい。
サクラエディタの設定 ＞ タイプ別設定 の各タブでインポートして下さい。
COLファイル：カラー タブ
KWDファイル：キーワードヘルプ タブ
RKWファイル：正規表現キーワード タブ

kotlinのタイプ別設定はあらかじめ作成しておいて下さい。
